## To Run The Project
Within the top level directory run the following commands:
```shell
./install.sh
```

```shell
./setup.sh
```

```shell
./run.sh
```

If any of them fail you may need to run:
```shell
chmod u+x ./install.sh ./run.sh ./setup.sh
```

## Ports
- Frontend: `5173`
- Backend: `8000`

## Package Managers
- `npm` for the frontend
- `pipenv` for the backend
